Assignment 2:

This code is used to establish a client-server architecture using a message oriented communication protocol to implement a file server. 
The operations that are performed are UPLOAD, DOWNLOAD, DELETE and RENAME. There is also a multi-threaded support using which multiple client can connect to the server.

Steps to execute:
1. The server folder under Assignment 2 has the server code and a Files folder. The files folder is used for all file operations on server side.
2. Change directory to server. Compile server code using the command javac Server.java
3. Run server code using java Server

4.The client folder under Assignment 2 has the client code and a Files folder. The files folder is used for all file operations on client side.
5. Change directory to client. Compile server code using the command javac client.java
6. Run server code using java Client
