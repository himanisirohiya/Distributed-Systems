Assignment 1:

This code is used to establish a client-server architecture using a message orientd communication protocol to implement a file server. 
The operations that are performed are UPLOAD, DOWNLOAD, DELETE and RENAME.

Steps to execute:
1. The server folder under assignment 1 has the server code and a Files folder. The files folder is used for all file operations on server side.
2. Change directory to server. Compile server code using the command javac Server.java
3. Run server code using java Server2

4.The client folder under assignment 1 has the client code and a Files folder. The files folder is used for all file operations on client side.
5. Change directory to client. Compile server code using the command javac Client.java
6. Run server code using java Client2
